Welcome to HSCC 2011

Join us for an exciting conference dedicated to advancing computer science education at the high school level. Connect with educators, students, and industry professionals.

Conference Sessions

Session 1-1

Introduction to Programming - Learn fundamental programming concepts and problem-solving techniques.

Session 1-2

Web Technologies - Explore HTML, CSS, and JavaScript for building modern web applications.

Session 2-1

Data Structures and Algorithms - Master essential data structures and algorithmic thinking.

Session 2-2

Object-Oriented Programming - Understanding OOP principles and design patterns.

Session 3-1

Introduction to Algorithm Design - Learn fundamental concepts and practical approaches to teaching algorithms.

Session 3-2

Advanced Topics in Computer Science Education - Explore innovative teaching methodologies and curriculum development strategies.

Session 3-3

Web Development for Beginners - Discover how to introduce students to modern web technologies.

Session 4-1

Data Structures in Practice - Practical applications of data structures in computer science education.

Session 4-2

Programming Languages Overview - Comparing different programming languages for educational purposes.

Session 4-3

Project-Based Learning - Implementing effective project-based learning in CS classrooms.

About HSCC

The High School Computer Conference brings together educators, students, and industry professionals to advance computer science education.
